ยกเลิกไม่ต้องใช้ ชุด Source code นี้
เพราะให้อิง SourceCode จากการ Install pip install gpt4all เพราะจะมีการปรับ Version การทำงานบน PIP Package อยู่เสมอ


> Version นี้จะมีปัญหาในการตอบคำถาม ไม่ตรงกับคำตอบ ตาม Prompt ที่ส่งไปให้
> มีการตอบคำถามที่พยายาม อธิบายคำตอบ และคำอธิบายก็มีความเพี้ยนมาก
> ถามคำถามเดิมซ้ำๆ  ได้คำตอบที่ผิดเพี้ยนไป และ ถูกบ้าง ผิดบ้าง ในบางครั้งเพี้ยนไปเลย
> ถึงแม้จะปรับค่า Temp แล้วแต่ก็ยังมีความเพี้ยนของคำตอบเช่นเดิม และ ชอบมี Explanation: .... (หรือพยายามอธิบาย)
